Kebijakan Privasi

TodoListApp (“Aplikasi”) dikembangkan oleh Nur Rizal Nadif Fikri (“kami”). Kebijakan ini menjelaskan bagaimana Aplikasi menangani data Anda.

Pembaruan terakhir: 26 Agustus 2025

Ringkasan

Aplikasi ini adalah aplikasi daftar tugas (to-do) sederhana. Kami tidak mengumpulkan atau membagikan data pribadi kepada pihak ketiga. Data to-do Anda disimpan lokal di perangkat (mis. melalui penyimpanan lokal seperti TinyDB/Kodular) dan tidak dikirim ke server kami.

Data yang Diproses

  • Data yang Anda masukkan (judul/isi tugas): disimpan lokal di perangkat Anda.
  • Data pribadi (nama, email, lokasi, kontak, foto, dll.): tidak dikumpulkan.
  • Berbagi data ke pihak ketiga: tidak ada.

Penyimpanan & Retensi

Semua data to-do disimpan lokal di perangkat. Anda dapat menghapus data kapan saja dengan menggunakan fitur “hapus tugas”, “reset” jika tersedia, menghapus data aplikasi di pengaturan sistem, atau mencopot pemasangan Aplikasi.

Perizinan Aplikasi

Aplikasi versi ini tidak meminta izin sensitif (kamera, mikrofon, lokasi, SMS, kontak, dll.). Jika di masa depan kami menambahkan fitur yang membutuhkan izin tersebut, izin akan diminta secara eksplisit di dalam aplikasi dan kebijakan ini akan diperbarui.

Keamanan

Kami merancang Aplikasi agar data to-do tetap berada di perangkat Anda. Meski tidak ada metode yang sepenuhnya aman, kami meminimalkan risiko dengan tidak mentransmisikan data ke server pihak kami.

Anak-Anak

Aplikasi tidak ditujukan untuk anak di bawah 13 tahun. Kami tidak mengetahui mengumpulkan data anak. Jika orang tua/wali percaya ada data anak yang tersimpan atau terkirim, hubungi kami untuk penanganan.

Perubahan Kebijakan

Kami dapat memperbarui kebijakan ini dari waktu ke waktu. Perubahan material akan diumumkan melalui pembaruan pada halaman ini dengan tanggal pembaruan terbaru.
